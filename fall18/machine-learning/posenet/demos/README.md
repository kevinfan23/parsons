# Pose-Controlled Web Interface with PoseNet

## Setup

cd into the project directory:

```sh
cd DIR_NAME
```

Install dependencies and prepare the build directory:

```sh
yarn
```

To watch files for changes, and launch a dev server with Yarn:

```sh
yarn watch
```
